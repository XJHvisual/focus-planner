# 专注规划器 v3.0

考研目标管理工具 — 把大目标拆成每一天的行动。

## 功能

| 模块 | 功能 |
|------|------|
| 🎯 目标拆解 | 智能拆解向导 → 五级目标体系（长期/年度/月度/周/日），树形展示、搜索、删除 |
| 📝 今日任务 | 从目标一键添加任务，双击标记完成，实时进度 |
| ⏱ 专注计时 | 15/25/45/60/90/120 分钟可选，暂停/继续/结束，完成后提示"可以去续火花" |
| 📊 数据统计 | 目标数、完成率、今日/总专注时长、历史记录 |

## 运行

```bash
pip install pyinstaller    # 打包时需要
python main.py             # 直接运行
python build_exe.py        # 打包成 exe
```

打包后的 exe 在 `dist/FocusPlanner.exe`。

## 数据

所有数据保存在 `data/` 目录：
- `goals.json` — 目标数据
- `tasks.json` — 任务数据
- `records.json` — 专注记录
- `focus_log.json` — 每日专注时长汇总
